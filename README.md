# Kirjasto - Helmet-kirjaston hakusovellus

Expo-pohjainen React Native -mobiilisovellus Helsingin kirjaston (Helmet) aineistojen hakemiseen Finnan avoimen rajapinnan kautta.

## Asennus

```bash
npm install
```

## Käyttö

Käynnistä Expo-kehityspalvelin:

```bash
npm start
```

Tai suoraan laitteelle:
- iOS: `npm run ios`
- Android: `npm run android`
- Web: `npm run web`

## API-testaus

Testaa Finnan API:n toimivuus:

```bash
npm test
```

## API-viittaus

Sovellus käyttää Finnan avointa rajapintaa:
- API-osoite: https://api.finna.fi
- Dokumentaatio: https://www.avoindata.fi/data/fi/dataset/helmet-kirjastojen-aineistoluettelot

## Teknologiat

- Expo ~51.0.0
- React Native 0.74.5
- React 18.2.0
